## 第一讲 · 人工智能概述

### A. 浅层学习 vs 深度学习

核心范式差异：**分段流水线 vs 端到端学习**。

| 维度 | 浅层学习 | 深度学习 |
|---|---|---|
| 流程 | 人工设计的多阶段流水线 | 输入→网络→输出，一气呵成 |
| **CV 例子** | 像素 → SIFT 特征点 → 视觉词典 → BoW → 金字塔 → SVM | 像素 → CNN → 分类 |
| **NLP 例子** | 词性标注 → 句法树 → 语义角色 → 翻译 | Encoder → Decoder（Seq2Seq） |
| 特征 | 人工设计（语言学/视觉特征工程） | 自动学习 |
| 优化 | 每段独立优化 | 全网络联合优化 |
| 代表突破 | 2012 前的主流 | AlexNet (2012)、Transformer (2017) |

**核心洞察**：2012 年后 CNN 取代传统 CV 流水线，2014 年后 Seq2Seq 取代基于句法的统计机器翻译——本质都是"端到端联合优化"碾压"人工拼接管线"。

### B. 机器学习的本质

```
机器学习 = 给定一堆 (x, y) 样本，自动找出能把 x 翻译成 y 的函数 f
y = f(x)
```

三步流程：
1. 从原始数据中提取特征
2. 学习映射函数 f
3. 通过 f 把新数据映射到语义空间

### C. 机器学习四大流派

| 类别 | 数据特点 | 任务 | 学习信号 |
|---|---|---|---|
| **监督学习** | 有标签 | 分类、回归 | 标准答案 |
| **无监督学习** | 无标签 | 聚类、降维 | 数据自身结构 |
| **半监督学习** | 少量有标签 + 大量无标签 | 同上 | 标签 + 数据结构 |
| **强化学习** | 状态-动作-奖励 | 决策、控制 | 环境反馈 |

### D. 监督学习的损失函数

```
训练映射函数 f，使得 f(xᵢ) 预测结果尽量等于 yᵢ
```

训练目标（核心公式）：
$$\min_{f} \sum_{i=1}^{n} \text{Loss}(f(x_i), y_i)$$

### E. 经验风险 vs 期望风险（过拟合的数学定义）

| 风险 | 公式 | 含义 |
|---|---|---|
| **经验风险**（ERM） | $\frac{1}{n}\sum \text{Loss}(y_i, f(x_i))$ | 训练集上的平均损失（能算的） |
| **期望风险** | $\int \text{Loss}(y, f(x)) P(x,y) dxdy$ | 真实分布上的损失（真正想要的） |

**四象限诊断表**：

| 经验风险（训练集） | 期望风险（测试集） | 状态 |
|---|---|---|
| 小 ✅ | 小 ✅ | 泛化能力强（理想） |
| 小 ✅ | 大 ❌ | **过学习**（模型过于复杂） |
| 大 ❌ | 大 ❌ | **欠学习**（模型太弱） |
| 大 ❌ | 小 ✅ | "神仙算法" 或 "黄粱美梦"（基本不存在） |

### F. 判别模型 vs 生成模型

| 维度 | 判别模型 | 生成模型 |
|---|---|---|
| 学什么 | $P(Y\|X)$ 或 $f(X)$ | $P(X, Y)$ 联合分布 |
| 关注点 | 决策边界 | 数据如何产生 |
| 比喻 | 给鱼（警察判案） | 给渔（画家先理解世界） |
| 能力 | 只能判断 | 能判断 + **能生成** |
| 代表 | 神经网络/SVM/逻辑回归 | 朴素贝叶斯/HMM/**GAN/Diffusion/GPT** |

**关键公式**（贝叶斯定理）：
$$P(Y|X) = \frac{P(X|Y) \cdot P(Y)}{P(X)}$$

### G. 神经元 · 激活函数 · 前馈网络

**神经元数学模型**（深度学习的"原子"）：
1. 加权求和：$\text{In} = \sum w_i \cdot a_i$
2. 非线性激活：$\text{Out} = g(\text{In})$

**三大常用激活函数**：

| 名字 | 公式 | 输出范围 | 主要问题 |
|---|---|---|---|
| Sigmoid | $\frac{1}{1+e^{-x}}$ | (0, 1) | 梯度消失，非零均值 |
| Tanh | $\frac{2}{1+e^{-2x}}-1$ | (-1, 1) | 仍有梯度消失 |
| **ReLU** | $\max(0, x)$ | [0, ∞) | Dying ReLU（但当今主力） |

**为什么必须有非线性**：若 $g$ 是线性，多层网络退化为单层矩阵乘法，连 XOR 都学不会。

**前馈神经网络（FNN）特征**：
- 信号单向流动（输入 → 隐藏 → 输出）
- 层间全连接、层内不连接
- 是 CNN / RNN / Transformer 的祖宗

**感知机（历史教训）**：
- 单层无隐藏层 → 只能学线性分类
- 经典反例 XOR → 引发 1969-1986 第一次 AI 寒冬
- 解药：加隐藏层 + 非线性激活 → 多层感知机 (MLP)

## 第二讲 · 机器学习基础（v4 游戏严格对标核心）

### A. 基本概念（PPT 第 1-10 页）

> **模型 = 算法 + 参数**

| 概念 | 定义 | 例子 |
|---|---|---|
| **算法** | 抽象的"流派" | 线性回归 `y = a + bx` |
| **参数** | 具体的数值 | a=80, b=2 |
| **模型** | 算法 + 具体参数 | weight = 80 + 2 × height |
| **超参数** | 参数以外的所有 decision | 学习率、复杂度、特征选择… |

**超参数的三大分类**：

| 分类 | 包含 |
|---|---|
| 数据准备 | 特征工程与选择 |
| 模型设定 | 算法、模型架构 |
| 模型训练 | 优化器、学习率、训练周期 |

### B. 如何得到参数（PPT 第 11-12 页）

| 任务 | 解决方案 | 关键目标 | 行为主体 |
|---|---|---|---|
| 训练参数 | **梯度下降** | 损失函数最小 | 🤖 计算机 |

**梯度下降本质 = 试错**：
1. 随机猜一组参数 (b₀, b₁)
2. 算损失 L
3. 朝"L 变小"的方向挪一小步
4. 重复直到不能再小

**损失函数**（三种形式，PPT 主推第三种）：
- $L = f(y - \hat{y})$（差值）
- $L = f(|y - \hat{y}|)$（绝对值）
- **$L = f((y - \hat{y})^2)$** ← MSE 均方误差（平方好求导，主流）

**梯度下降的牛之处**：不需要知道最低点在哪，只需要知道下一步往哪走。

### C. 如何得到超参数（PPT 第 13 页 · v4 游戏核心）

**数据三分**：

```
100% 数据
  ↓
切成三块
  ├─ 训练集 80% （平时练习 · 训练参数用）
  ├─ 验证集 10% （模拟考·调超参 · 可多碰）
  └─ 测试集 10% （高考·真实评估 · 只能碰一次）
```

| 任务 | 解决方案 | 关键目标 | 行为主体 |
|---|---|---|---|
| 选择超参数 | **数据拆分** | 性能指标最高 | 👤 人类 |

**为什么三分**（字幕原话）：
> "测试集理论上只能碰一次，就像高考只能参加一次。"
> "验证集倒是可以多碰几次，但是尽量少碰。"

### D. 调整超参数的方法（PPT 第 17-19 页 · 防卷神器）

PPT 红字装备清单：

| 装备 | 治欠拟合 | 治过拟合 |
|---|---|---|
| 增加模型复杂度 | ✅ | ❌ |
| 增加特征数量 | ✅ | ❌ |
| 增加训练周期 | ✅ | ❌ |
| **使用早停 Early Stopping** | ❌ | ✅ |
| 增加训练数据 | / | ✅ |
| **数据增强 Data Augmentation** | / | ✅ |
| **L1 / L2 参数管制 Regularization** | / | ✅ |
| **Dropout 随机失活** | / | ✅ |

**L1 / L2 正则化对比**：

| | L1 | L2 |
|---|---|---|
| 损失项 | $\text{Loss} + \lambda \sum \|w_i\|$ | $\text{Loss} + \lambda \sum w_i^2$ |
| 效果 | 让某些权重变 0（**稀疏**） | 让所有权重变小（**收缩**） |
| 几何 | 菱形约束区 | 圆形约束区 |

**Dropout**：训练时随机让一部分神经元"失活"（典型 p = 0.5），避免过度依赖单个神经元。

### E. 学习曲线 vs 验证曲线（PPT 第 15-16 页）

| | 学习曲线 | 验证曲线 |
|---|---|---|
| X 轴 | 训练数据数量 | 超参数取值范围 |
| Y 轴 | 性能指标 | 损失或性能指标 |
| 用途 | 诊断**欠拟合 vs 过拟合** | **超参数选择** |

### F. 机器学习的任务类别（PPT 第 22-29 页）

```
ML 任务
├─ 监督学习 (supervised)
│   ├─ 回归 (regression) → linear regression / LSTM
│   └─ 分类 (classification)
│       ├─ 文本分类 / NER
│       ├─ 图片分类 / 物体检测 / 图片分割
│       └─ 算法：logistic / 随机森林 / gradient boosting
└─ 无监督学习 (unsupervised)
    └─ 聚类 (clustering) → k-means
```

**业务案例对应**：

| 场景 | 任务类型 |
|---|---|
| 简历筛选 | 分类预测 |
| 招聘规划 / 仓储备货 / 信贷额度 | 数值预测（回归） |
| 信用评分 | 数值预测 + 分类预测 |
| 欺诈检测 | 聚类 + 分类预测 |

### G. 分类任务性能指标（PPT 第 47-53 页）

**核心 4 大指标**：

| 指标 | 公式 | 直觉含义 |
|---|---|---|
| 准确率 Accuracy | Correct / Total | 整体对的比例 |
| 精确率 Precision | TP / (TP + FP) | 你说"对"的里面真对的比例 |
| 召回率 Recall | TP / (TP + FN) | 真对的里面被你抓到的比例 |
| **F1** | 2·P·R / (P+R) | **水桶理论**——P/R 短板决定 |

**注意**：
- 数据高度不平衡时，**准确率会骗人**（PPT 第 41 页：全猜成"该钓"也能达到 80%）
- F1 符合木桶原理——精确率和召回率有一个低就拉胯

**混淆矩阵**：

```
              预测：正  预测：负
实际：正        TP        FN
实际：负        FP        TN
```

**多类别聚合**：
- **宏观平均 macro** = (Pa + Pb + ... + Pd) / 4（不考虑类别样本量）
- **微观平均 micro** = 把所有 TP/FP/FN 加起来一起算
- **加权平均 weighted** = Σ (Nᵢ/Nsum) · Pᵢ（按样本量加权）

|---|---|---|---|
| 训练参数 | 梯度下降 | 损失函数 | 🤖 计算机 |
| 选择超参数 | 数据拆分 | 性能指标 | 👤 人类 |
